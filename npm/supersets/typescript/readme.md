### Instalação

1. *Instalação por terminal de comandos*
    ```
    npm install -D typescript
    ```
2. [**C***ompilar type script*](cnpx.md) para: [*configurações*](tsconfigjson.md)
    ```
    npx tsc --init
    ```
    1. npx -- node package e**X**ecutor
    2. tsc -- type script **compiler**
