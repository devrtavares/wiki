# git init

Após instalar o git para iniciar o controle de versão basta seguir para o diretório dos arquivos a serem controlados e executar:

```
git init
```

*Exemplo utilizando o cmd do windows*:
```
cd /d Desktop
mkdir "novo-modulo"
cd /d "novo-modulo"
git init
```
