### eslint integrado ao visual studio code

- ***instalação via terminal:***
    ```bash
    code --list-extensions
    code --install-extension dbaeumer.vscode-eslint
    ```
    - remoção:
        ```bash
        code --uninstall-extension dbaeumer.vscode-eslint
        ```
