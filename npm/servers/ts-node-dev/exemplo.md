## *ts-node-dev - exemplo*


- package.json 

    ```json
    "dev": "ts-node-dev --respawn --transpile-only --ignore-watch node_modules --no-notify src/server.ts"
    ```

    - **server.ts**
        ```javascript
        import express from "express";

        const app = express();

        app.get(`/`, (request, response) => {
        return response.json({ message: `hello World` });
        });

        app.listen(3333);

        ```
        - run
            ```bash
            npm run dev
            ```
